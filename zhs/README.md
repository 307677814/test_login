# 智慧树

# 用法

# 一，刷视频
`ChromeVersionPassVideo.js`里面的内容直接粘贴到谷歌浏览器控制台
- 1.打开视频，点击播放
- 2.按键盘F12打开Chrome浏览器控制台
- 3.找到console窗口，复制代码到窗口，粘贴回车
#### 已实现的功能:
- 自动下一集
- 自动1.5倍速
- 自动填写弹窗
- 静音播放
